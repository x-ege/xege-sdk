主页 / 库函数目录 / 文字输出 / ege_drawtext

功能：

在指定位置格式化输出文本（使用 ARGB 颜色，由 setcolor() 或 settextcolor() 指定）。

声明：

void ege_drawtext(
    const char* format,
    float x,
    float y,
    PIMAGE pimg = NULL
);
void ege_drawtext(
    const wchar_t* format,
    float x,
    float y,
    PIMAGE pimg = NULL
);


参数：

format
格式字符串。

x
文本输出位置的 x 坐标(文本实际位置与文本对齐方式、文本倾斜角度等相关)。

y
文本输出位置的 y 坐标(文本实际位置与文本对齐方式、文本倾斜角度等相关)。

pimg
要输出文本的目标图像（如果为 NULL，则输出到窗口）。

返回值：

（无）

示例：
ege_drawtext("text", 10.0f, 60.5f);
